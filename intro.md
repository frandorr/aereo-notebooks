# AerEO Tutorials

This is an executable book of tutorials for [AerEO](https://github.com/frandorr/aereo), a modular satellite data discovery, extraction, and processing framework.

## How to use this book

- **Read online**: browse the rendered chapters below.
- **Run on Binder**: click the rocket icon 🚀 at the top of any page to launch an interactive session on [mybinder.org](https://mybinder.org).
- **Run live in the browser**: click the "Live Code" button (powered by [Thebe](https://thebe.readthedocs.io)) to execute cells without leaving the page.
- **Run locally**: clone this repo, install the dependencies, and run the notebooks with JupyterLab.

## Notebooks

The tutorials cover Sentinel-2, VIIRS, Sentinel-3 OLCI, GeoTessera, GOES-19 ABI, and multi-constellation workflows.

## Authentication

Notebooks that use NASA data (VIIRS, Sentinel-3, etc.) require [earthaccess](https://earthaccess.readthedocs.io) credentials. Configure a `~/.netrc` file before running those examples — see the [earthaccess authentication guide](https://earthaccess.readthedocs.io/en/latest/user/howto/authenticate/).
